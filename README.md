# 📉🛡️ CreditRiskAI - Intelligent Loan Delinquency Management Agent

**Advanced Agentic AI System for Credit Risk & Collections**

An intelligent, ethical, and production-ready **Agentic AI** that helps financial institutions reduce credit card and loan delinquency through smart reasoning, memory, ethical guardrails, and automated tool execution.

## ✨ Key Features

- **Advanced Reasoning**: Simulated LLM-style natural language decision making
- **Memory System**: Remembers past interactions with each customer
- **Ethical Guardrails**: Protects vulnerable customers (seniors, unemployed, young)
- **Tool Execution**: Can send SMS reminders, offer payment plans, and escalate cases
- **Reflection**: Reviews its own previous actions before making new decisions
- **Interactive Dashboard**: Search, history view, and side-by-side customer comparison
